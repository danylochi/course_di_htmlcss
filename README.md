# Curso Dio Html + Css

Projeto front-end estatico (HTML + CSS) criado como exercicio pratico, com foco em estrutura de pagina institucional simples.

## Visao geral

Este repositorio contem uma pagina unica (`index.html`) com secoes de:

- Servicos
- Clientes
- Parceiros
- Time
- Contato

O projeto nao usa framework, bundler ou dependencias externas de build.

## Stack e arquitetura

- HTML5
- CSS3
- Assets locais em `img/`

Arquivos principais:

- `index.html`
- `css/style.css`
- `img/*`

## Estrutura do projeto

## Como executar localmente

Como e um projeto estatico, basta abrir o arquivo `index.html` no navegador.

## Escopo funcional atual

- Navegacao superior com links para secoes da pagina.
- Blocos com imagens para servicos, clientes, parceiros e time.
- Secao de contato com links `mailto:` e `tel:`.
